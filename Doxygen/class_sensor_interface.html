Detailed Description

Sensorler icin bir arayuz sinifi.

Bu soyut sinif, turetilen sensor siniflarinda implement edilmesi gereken temel metodlari tanimlar. Bu sinif, sensorlerin tipini tanimlama, veri alma ve guncelleme gibi islevleri icermektedir.
